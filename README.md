Heavy/Light Claude Code Modes
These files are what I use to launch claude code on my mac but it asks me if i want a light or heavy session, light has almost all plugins disbaled and heavy keeps everything enabled.
I use this for token optimisation, using a main claude session with a claude model running and a secondary session running a free model from open router (see launch-claude.sh) and in switch-mode.sh is where the session mode comes in.
the .md file was made by claude after it finished creating the switch mode file.
